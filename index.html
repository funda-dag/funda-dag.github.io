Stichpunkte und Notizen zu Angular!

Komponente einbinden:

    Komponenten können in Komponenten eingebunden werden:
  • app.component.ts :

    • @Component({
    selector: 'app-select-field',
    templateUrl: './select-field.component.html',
    styleUrls: ['./select-field.component.scss'],
    })

  • Selector aus ts in html einbinden
    • app-select-field [values]="residenceStatusValues" [fieldControl]="get('firstResidenceStatus')!">
    Komponente erstellen wenn:
  • etwas öfters vorkommt
  • viel Logik beinhaltet

Kommunikation zwischen Komponenten

    Kommunikation zwischen Komponenten
  • Input(): Eingabe
    Der Komponente wird ein Wert von der Elternkomponente gegeben

  • Output(): Rückgabe
    Der Wert in der Komponente wird an die Elternkomponente zurückgegeben

  • emit
    Elternkomponente etwas zurück geben

    Komponenten haben einen Lebenszyklus
  • ngOnInit
    • wird jedesmal aufgerufen wenn Komponente erstellt wird
    hier ist nichts zu sehen
  • ngAfterViewInit
    • wird aufgerufen wenn HTML fertig ist
    Nutzer sieht die Benutzeroberfläche
  • ngOnDestroy
    • wird aufgerufen wenn Komponente stirbt (z.B. Seitenwechsel)
    Sinnvoll in Formularen: Verlassen vom Formular -> Beobachtung beenden

    Bindings
  • [ ]
    • Übergabe
    Wert von EK wird an KK übergeben
    EK bekommt Änderungen nicht mit


  • ( )
    • Empfang
    übergibt nichts
    empfängt etwas


  • [( )]
    • empfangen und zurückgeben


  • {{ }}
    • String ausgeben
    Wert an HTML übergeben und anzeigen
    Text ausgeben
    Komponente etwas übergeben


    Directives
  • *ngFor
    • alle Elemente eines Arrays durchlaufen
  • *ngIf
    • Wenn true: dieses Element existiert in HTML
    Wenn false: dieses Element existiert nicht!

    true -> false:
    Komponente wird zerstört!
    (Nicht mehr in HTML drin)
    false -> true:
    Komponente wird erstellt!
    (ngOnInit, ngAfterViewInit, ngOnDestray wird aufgerufen)


    Vergleich 'Display:none' und 'ngIf'
  • Display:none;
    • Element noch da, ABER keine Anzeige
  • ngIf
    • Element komplett WEG!
